# test-app

## Random Java Web App

This is simply a demo app used to test Jenkins pipelines.

See [my other project](https://github.com/Direside/jenkins-test-environment) for it's use-case.

### Build
```bash
cd java-web-app
mvn clean verify
```

### Test
```bash
cd java-web-app
mvn test
```
